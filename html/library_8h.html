Detailed Description

A stack é uma estrutura estrutura de dados linear na ordem 'Last In First Out' (LIFO), sendo nessa ordem que são realizadas e caracteriza um empilhamento de dados.

Neste ficheiro tem-se como objectivo a implementação da stack.
