\doxysubsection{Detailed Description}
A stack é uma estrutura estrutura de dados linear na ordem \textquotesingle{}Last In First Out\textquotesingle{} (LIFO), sendo nessa ordem que são realizadas e caracteriza um empilhamento de dados.

Neste ficheiro tem-\/se como objectivo a implementação da stack.
